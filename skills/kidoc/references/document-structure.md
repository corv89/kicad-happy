# Document Structure Reference

Section templates for each document type generated by kidoc.

## Hardware Design Description (HDD)

1. Front Matter — title, revision history, TOC, acronyms
2. System Overview — architecture block diagram, functional description, key specs
3. Power System Design — power tree, per-rail specs, budget, sequencing, decoupling
4. Signal Interfaces — per-bus sections (I2C, SPI, UART, CAN, USB, Ethernet, etc.)
5. Analog Design — voltage dividers, filters, opamps, ADC, sensors, crystals, SPICE results
6. Thermal Analysis — hotspot table, per-component Tj, thermal via strategy
7. EMC Considerations — risk score, findings by category, per-net scores, PDN impedance
8. PCB Design Details — layer stackup, dimensions, impedance control, routing stats, DFM
9. Mechanical / Environmental — board outline, mounting holes, temperature range
10. BOM Summary — full BOM table, cost summary, lifecycle status
11. Test and Debug — test points, debug interfaces, production test strategy
12. Compliance and Standards — applicable standards, EMC test plan, regulatory coverage
13. Appendices — full schematic sheets, PCB views, 3D renders, Gerber summary

## CE Technical File

1. Product Identification — name, model, intended use, specifications
2. Essential Requirements — EU LVD + EMC + RoHS mapping to design evidence
3. Harmonized Standards — EN 55032, EN 55035, EN 62368-1, etc.
4. Risk Assessment — electrical hazards, overheating, EMI, mechanical
5. EMC Test Plan — from analyze_emc test plan output
6. Safety Assessment — from thermal and protection circuit analysis
7. Technical Construction File — schematics, PCB details, BOM
8. Declaration of Conformity — pre-filled DoC template
9. Appendices — schematics, test reports

## Interface Control Document (ICD)

1. System Context — system-level block diagram showing interfaces
2. Interface List — summary table of all interfaces
3. Connector Details — per-connector: pinout, signal levels, protection
4. Signal Definitions — per-signal: direction, voltage, protocol, timing
5. Electrical Characteristics — voltage levels, impedance, current limits
6. Mechanical Interface — connector type, mounting, mating force
7. Appendices — full pinout tables

## Design Review Package

1. Review Summary — risk scores, critical findings count
2. Critical Findings — highest-severity issues requiring resolution
3. Schematic Review — signal analysis findings
4. PCB Review — layout findings, DFM issues
5. EMC Assessment — EMC risk summary
6. Thermal Assessment — thermal findings
7. BOM Review — lifecycle, sourcing, missing MPNs
8. Action Items — resolution table with owners and dates

## Manufacturing Transfer Package

1. Assembly Overview — board description, assembly notes
2. BOM — full BOM with MPN, distributor PNs, quantities
3. PCB Fabrication Notes — stackup, material, finish, impedance, DFM constraints
4. Assembly Instructions — soldering, component placement, special handling
5. Test Procedures — production test steps, pass/fail criteria
6. Inspection Criteria — visual inspection, X-ray requirements
7. Appendices — Gerber summary, assembly drawings
